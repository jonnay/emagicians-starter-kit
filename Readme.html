3 Five 'W's and 2 'Y's.

3.1 Who am I?

I'm Jonnay!

I've been using Emacs since 2007. I make Music, Art and Video Blogs. I'm a front-end narwhal (a Canadian coder who can do UX and design as well). Professionally I have used many languages—including P/LB—but my favorite is Scheme, and my current is ruby.

3.2 What is this? A starter kit for magicians.

The Emagicians Starter Kit is a set of org-mode files that are automagikally tangled into Emacs Lisp on startup.

I got tired of my old .emacs file and poorly maintained collection of extensions. It was working less and less well the more and more time went on and on.

But it's more than that. It's a set of lessons learned, ramblings of a lisp fanatic, and something of a personal great work.

3.3 Why?

I Love the idea of the starter kits, but I found they made assumptions and just wasn't fit for what I needed. I also have a bit of a "not-invented-here" problem. It's minor and I cope pretty well. However with regards to emacs: (and to paraphrase Douglas McArther Shaftoe from the Neal Stephenson novel Cryptonomicon.)

This is my Emacs. There are plenty like it, but this one is mine.

Additionally, there is real value in people explaining their emacs setup, especially in easily digestible forms.

3.4 How

Everything is in Org-Mode. If you want to join in all the fun, there is an ./Emagician-Install.html file that you can follow along with. It will get you moving into basic configurations, all the way through testing the result before making the emagician starter kit your actual init file.

3.5 When

March 2012. It got commited to a git repo later that year.
